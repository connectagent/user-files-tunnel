# Quick Start

1. [Download the source files](https://github.com/connectagent/user-files-tunnel/archive/refs/heads/main.zip)
2. Unzip the archive.
3. Run `uftunnel.py`.


The utility automatically opens a web app where you need to sign in and approve the connection. 
The connection details are saved to a file, which you can delete to reset (`uftunnel.py --reset`) the setup and run the command again.
